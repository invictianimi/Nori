## Color Palette

Primary accent: **Cyan / Ice Blue** (from logo)

Use this accent color for:
- Headings in generated documents
- Status indicators (OK state)
- Section dividers

---

## Typography

For markdown documentation:
- Use standard GitHub-flavored markdown
- Headings: `#`, `##`, `###` hierarchy
- Code blocks: triple backticks with language specifiers
- Tables: standard markdown table syntax

---

## Document Templates

All NORI-generated documents should:
- Use clean, minimal styling
- Maintain readability over decoration
- Follow consistent heading hierarchy
- Include NORI logo reference where appropriate (reports, presentations)
